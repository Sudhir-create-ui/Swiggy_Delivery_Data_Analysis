# Swiggy_Delivery_Data_Analysis
Project Overview

This project analyzes 8,600+ Swiggy delivery orders to uncover insights into delivery performance, customer behavior, and operational efficiency. The goal is to transform raw delivery data into meaningful insights that can support data-driven decision-making.
____________________________________________________________________________________________________________________________________________________________________________________________________________________
**Dataset Details**
Records: 8,600+ rows
Format: CSV
____________________________________________________________________________________________________________________________________________________________________________________________________________________
**Key Fields:-**
Order ID
City
Restaurant
Delivery Time (minutes)
Distance (km)
Order Amount
Order Status
Customer Rating
Order Date
____________________________________________________________________________________________________________________________________________________________________________________________________________________
**Tools & Technologies**
Excel – Data cleaning, validation, and preprocessing
SQL – Data analysis and querying
Power BI – Interactive dashboards and visualizations
____________________________________________________________________________________________________________________________________________________________________________________________________________________
**Analysis Performed**
Delivery time analysis by city and distance
Order status distribution (Delivered, Delayed)
Restaurant and city-wise performance comparison
Customer rating analysis and correlation with delivery time
____________________________________________________________________________________________________________________________________________________________________________________________________________________
**Key Insights**
Delivery time increases significantly as distance increases beyond a certain range
Some cities show higher cancellation and delay rates than others
Faster deliveries generally receive higher customer ratings
____________________________________________________________________________________________________________________________________________________________________________________________________________________
**Dashboard Features (Power BI)**
KPI cards: Total Orders, Avg Delivery Time, Avg Rating
City-wise and restaurant-wise performance charts
Delivery time vs distance analysis
Order status breakdown with interactive filters
____________________________________________________________________________________________________________________________________________________________________________________________________________________
**Business Value**
Helps identify cities or restaurants with delivery delays
Supports optimization of delivery routes and partner allocation

Improves customer experience by linking delivery speed to ratings
